# 目的

SolidjsとFlaskによるアプリケーション作成
ずんだもんとしりとりをする

# 遊び方

```shell
$ docker compose build
$ docker compose up -d
```

[localhost:3000](localhost:3000)にアクセス

最初に「しりとり」と入力してゲームスタート.デバッグしていないので上手く動く保証がないです。
